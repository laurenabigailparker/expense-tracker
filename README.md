# Budget Planner & Expense Tracker

A responsive budgeting app that tracks income, expenses, and category spending with real-time updates. Includes persistent data storage using localStorage, dynamic chart visualization, and dark mode support.

## Features
- Add and delete expenses
- Categorize spending (Bills, Gas, Groceries, Fun, Savings)
- Set monthly income
- Calculate total expenses
- Show remaining money
- 50/30/20 budget recommendations
- Compare actual spending vs recommended budget
- Visual feedback (over budget vs on track)

## Tech Stack
- HTML
- CSS (Grid + Flexbox)
- JavaScript (Logic + DOM manipulation)

## How It Works
The app stores:
- income as a number
- expenses as an array of objects

Each expense contains:
- name
- amount
- category

The app calculates:
- total expenses
- remaining money
- recommended budget (50/30/20 rule)
- actual category totals
